Класс LinkedHashMap расширяет HashMap и поддерживает связанный список записей на Map в том порядке,
в котором они были добавлены (*.put()). То есть при итерировании LinkedHashMap элементы будут возвращены в
том порядке, в котором они были вставлены.

Можно создать LinkedHashMap, который возвращает элементы в том порядке, в котором они были в последний раз.

Конструкторы:
-> LinkedHashMap() - Этот конструктор создает стандартную ссылку LinkedHashMap.
-> LinkedHashMap(Map m) - Этот конструктор инициализирует LinkedHashMap с элементами из данного класса Map m.
-> LinkedHashMap(int capacity) -Этот конструктор инициализирует LinkedHashMap с заданной емкостью.
-> LinkedHashMap(int capacity, float fillRatio) - Этот конструктор инициализирует как емкость, так
и коэффициент заполнения. Значение емкости и коэффициент заполнения такие же, как для HashMap.
-> LinkedHashMap(int capacity, float fillRatio, boolean Order) - Этот конструктор позволяет указать, будут
ли элементы храниться в связанном списке по порядку вставки или по порядку последнего доступа.
Если Order = true, используется порядок доступа. Если Order = false, используется порядок вставки.

Методы помимо, унаследованных от его родительских классов, LinkedHashMap имеет следующие методы:
-> void clear() - Удаляет все отображения из этого Map.
-> boolean containsKey(Object key) - Возвращает true, если этот Map отображает один и более ключей
в указанное значение.
-> Object get(Object key) - Возвращает значение, на которое эта карта отображает указанный ключ.
-> protected boolean removeEldestEntry(Map.Entry eldest) - Возвращает true, если эта карта должна
удалить свою старшую запись.
