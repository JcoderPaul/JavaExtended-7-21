Интерфейс Set – это наследник Collection (расширяет интерфейс), который !!! НЕ СОДЕРЖИТ !!!
повторяющиеся элементы. Интерфейс Set содержит только методы, унаследованные из Collection,
и добавляет ограничение, которое запрещает повторяющиеся элементы.

Set не добавляет новых методов, только вносит изменения в унаследованные. В частности, метод add()
добавляет элемент в коллекцию и возвращает 'true', если в коллекции еще нет такого элемента.

Set также усиливает до предела имплементацию, т.е. перезаписывает поведение методов equals и hashCode,
позволяя сравнивать экземпляры Set, даже если их типы реализации различаются. Методы, объявленные в Set,
приведены ниже:

-> add( ) - Добавляет объект к коллекции.
-> clear( ) - Удаляет все объекты из коллекции.
-> contains( ) - Возвращает true, если указанный объект является элементом в коллекции.
-> isEmpty( ) - Возвращает true, если в коллекции нет элементов.
-> iterator( ) - Возвращает объект Iterator для коллекции, который может быть использован
для извлечения объекта
-> remove( ) - Удаляет указанный объект из коллекции.
-> size( ) - Возвращает число элементов в коллекции.
